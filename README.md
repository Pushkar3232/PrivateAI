# Private AI Project (Work in Progress)
## *Overview*
Private AI is an offline artificial intelligence system designed to prioritize user privacy and functionality. Built using LLaMA, it leverages modern web technologies for a user-friendly interface while ensuring all computations are performed locally. This project is under development, with approximately 50% completed.<br><br>

Technologies Used:- <br>
AI Model: LLaMA<br>
Backend: Python, Flask<br>
Frontend: HTML, CSS, JavaScript<br><br>
Features:-<br>
AI-Powered Interactions:<br>
Local AI processing with LLaMA for text generation and contextual responses.<br>
Web-Based Interface(Work in Progress):<br>
Interactive frontend using HTML, CSS, and JavaScript.<br>
Offline Functionality:<br>
No internet required for running the AI system. <br> <br>
