# Private AI Project (Work in Progress)
## *Overview*
Private AI is an offline artificial intelligence system designed to prioritize user privacy and functionality. Built using LLaMA, it leverages modern web technologies for a user-friendly interface while ensuring all computations are performed locally. This project is under development, with approximately 50% completed.

Technologies Used
AI Model: LLaMA
Backend: Python, Flask
Frontend: HTML, CSS, JavaScript
Features (Work in Progress)
AI-Powered Interactions:
Local AI processing with LLaMA for text generation and contextual responses.
Web-Based Interface:
Interactive frontend using HTML, CSS, and JavaScript.
Offline Functionality:
No internet required for running the AI system.
